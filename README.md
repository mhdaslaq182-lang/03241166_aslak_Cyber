# Cyber Awareness Hub

Complete cyber security application with 6 security tools:
- Password Strength Checker
- Phishing Email Scanner  
- VPN/Proxy Detector
- Malware/Phishing URL Scanner
- SSL Certificate Validator
- Data Breach Checker

## Features:
- Responsive design
- Dark/Light mode
- Interactive security tools
- Modern UI with Tailwind CSS
- Express server for production
